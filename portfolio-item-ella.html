EllaYour personal medical assitant

Hyland Hackathon 2021, Team SMSN

This is my first time as a hackathon participant. It was a team of four members. Ella is a telegram bot which acts as a personal medical assistant. The bot can analyse your symptoms(mental, physical) and accordingly suggest doctors and even provide links to book video consulation with the doctor. The bot can also analyse covid symptoms and give details of nearest vacant covid beds. The bot uses an API to fetch data of the covid beds. The user can also send a SOS message to the Ella Community asking help for oxygen cylinders, blood etc. For now, anyone can log into Ella Community via their google account. Covid bed details of only West Bengal can be fetched by the API at present.

I did some minor editing in the EC User Interface but mainly worked on the conversation flow for which we used Dialogflow. Dialogflow is a natural language understanding platform used to design and integrate a conversational user interface into mobile apps, web applications, devices, bots, interactive voice response systems and related uses.

Tech Stack Used

  • React
  • NodeJS
  • Firebase
  • Dialogflow
