Mochabrain Face detection webapp

Personal Summer Project

This is my first react project built while following Andrei Neogie's course on Web Development "Zero to Mastery". At first, the user has to register or sign in(if previously registered). The user will then be redirected to the homepage of the application.

The user can now submit a link and any human face in the image will be detected.

As we can see in the screenshot, the girl's face has been detected and the image count of the user has also been incremented.

Tech Stack Used

  • ReactJS
  • NodeJS
  • PostgreSQL
